### WAT Framework — Agentic AI Architecture
*Python · Claude API · MCP · Markdown Workflows*

Purpose-built agentic framework separating probabilistic AI reasoning from deterministic code execution for reliable multi-step automation.

- **Architecture:** Workflows (plain-language Markdown SOPs) → Agent (decision + orchestration layer) → Tools (Python scripts, deterministic)
- **Key problem solved:** 5-step AI pipelines compound error at ~90%/step (59% end-to-end success); WAT confines AI to decisions only, tools to execution — dramatically improving reliability
- **Self-improvement loop:** Failure → trace → fix tool → verify → update workflow; system gets stronger with each failure
- Used for automating multi-step data workflows, reporting pipelines, and scraping operations
